Handles all IOC events.
We use several sources for IOC interrupt:

  • on any rising edge on port RA5/4, RC5/4, RC3/6, RC7/RB7: read current.
  • on any falling edge on port RA5/4, RC5/4, RC3/6, RC7/RB7: read Back EMF.

Must wait until current has settled, reading right after PWM ON gives results much too low!

Todo:
  • Preliminary we stall and waste 2 (to 3) ms with delay. Just trigger a second interrupt (timer) after that time.
Note
Low pass filter could be initialized using the first reading to avoid exponential course at the beginning. Alternatively, the low pass can be omitted, then there is also no phase shift. If the evaluation is done by SW, noise has to be suitably ignored. Furthermore: two successive sampling moments cannot be a be a zero crossing - must be checked by algorithm.
