◆ adc_init()

void adc_init ( uint8_t  chs)

Initializes the 12-bit A/D Converter (right adjusted result).

Parameters
chsBinary code of voltage source, see datasheet: 40.7.8 ADPCH
  • ADC stop & disable, clock supplied by FOSC
  • ADRES+ADPREV data right-justified
  • no precharge
  • Double-Sample disable
  • Legacy mode, no filtering, ADRES->ADPREV
  • no math functions, never interrupt
  • VREF- = AVSS, VREF+ = internal ADFVR
  • Clock divider: Conversion clock = FOSC/32
  • ADC Positive Input = chs
  • External Trigger Disabled
  • Result = 0
  • FVR Gain is 2x (-> 2.048 V)
  • Enable FVR
  • Disable ADC interrupt
  • Clear interrupt flag
