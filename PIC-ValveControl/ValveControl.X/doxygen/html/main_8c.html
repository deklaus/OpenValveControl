◆ main()

void main ( void  )

This is the Main program.

  • Initialize the device

Main loop:

  • Measure VDD [0.01 V] (optional battery check)
  • Check for requests from ESP via RS232
  • MOVE command
  • HOME command (close direction)
  • IDLE (kind of scheduler: check status bits for pending jobs).
    • For sanity, the IOC INT is disabled and enabled only during MOVE and HOME.
    • Measure the motor current in idle mode (PWM is off) as a control feature.
    • Check the status flags for incoming requests from ESP and set main_state and parameters as needed.
